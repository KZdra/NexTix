# NexTix!
Website Ticketing System 
Made With Vuejs + Tailwind + element+

## Api
[Helpdesk-Api](https://github.com/KZdra/helpdesk-api)

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
